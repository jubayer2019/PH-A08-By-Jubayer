# 🛒 SunCart – Summer Essentials Store

SunCart is a modern, responsive eCommerce storefront focused on summer essentials — sunglasses, apparel, skincare, and beach accessories. It demonstrates a full-stack Next.js application with authentication, protected routes, and product management features.

---

## 🌐 Live Preview
👉 https://suncart-by-jubayer.vercel.app/

---

## 🎯 Purpose
- Provide a polished demo eCommerce store
- Showcase authentication and protected routes
- Demonstrate product listing and detail pages
- Build a responsive UI suitable for portfolio or production-ready demos

---

## ✨ Key Features

- 📱 **Responsive UI** — Fully optimized for mobile, tablet, and desktop
- 🛍️ **Product Catalog** — Dynamic product listing and detail pages
- 🔐 **Authentication System** — Email/password and Google login
- 🚧 **Protected Routes** — Secure pages for logged-in users only
- 👤 **User Profile Management** — Update name, avatar, and profile data
- ⚡ **Fast Performance** — Built with Next.js App Router
- 🎨 **Smooth UI Animations** — Enhanced user experience with motion effects
- 🗄️ **Database Integration** — Lightweight DB handling for users & products

---

## 📦 NPM Packages Used

- `next` — React framework for production
- `react`, `react-dom` — Core React libraries
- `better-auth` — Authentication & session management
- `drizzle-orm` — Type-safe ORM for database handling
- `better-sqlite3` — SQLite database engine
- `framer-motion` — Animations and transitions
- `lottie-react` — Lottie animation support
- `lucide-react` — Icon library
- `clsx` — Conditional className utility
- `tailwind-merge` — Tailwind class merging utility

## 🚀 Getting Started (Local Setup)

### 1. Install dependencies

npm install

### 2. Run development server

npm run dev

### 3. Open in browser

http://localhost:3000
